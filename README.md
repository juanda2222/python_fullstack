# Python flask fullstack application

## PUNTO 1

To install the applicattion localy under the "PUNTO 1" folder:

> python -version = 3.7
> pip install requirements.txt
> python setup.py

To test the aplication:
> python test.py

To run the aplication:
> python main.py
